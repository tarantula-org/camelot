# Code of Conduct

**Authority:** Avant Systems Canon (ASC-1.2)

This repository is a workspace for high-performance systems engineering. We value technical rigor, direct communication, and architectural integrity.

### Standards
* **Technical Objectivity:** Critiques must be focused on code quality, performance, and correctness. Ad hominem attacks result in immediate exclusion.
* **Scope:** This is a technical channel. Political, social, or off-topic discourse is strictly out of scope.
* **Hierarchy:** The Steering Committee (`@Steering`) retains final authority on all architectural decisions to preserve the kernel's vision.

### Enforcement
Violations should be reported directly to the Maintainers.
